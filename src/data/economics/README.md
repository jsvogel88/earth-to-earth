# Economic enrichment data

Generated by **`npm run enrich:economic`** from:

| Source | Data |
|--------|------|
| [GeoNames cities15000](https://download.geonames.org/export/dump/cities15000.zip) | Latitude, longitude, city population |
| [GeoNames countryInfo](https://download.geonames.org/export/dump/countryInfo.txt) | Country name → ISO2 |
| [World Bank Open Data](https://data.worldbank.org/) | GDP (current USD), country population, GDP per capita, urban % |

## Output files

- `country-economics.json` — keyed by ISO2
- `city-economics-by-geonameid.json` — keyed by GeoNames ID (joins to `world-cities.csv`)
- `enrichment-meta.json` — run stats and timestamp

## Derived city fields

- `gdpEstimateUsd` — country GDP × (city pop / country pop)
- `businessTravelers` — estimated from population and GDP per capita
- `airportCapacity` — estimated from population
- `regulatoryEase` / `geopoliticalStability` — proxies from income level and region

## Usage

```javascript
import { enrichWorldCityRecord } from './loadEconomics.js';
import { loadWorldCitiesEnriched } from '../worldCities.js';
import { loadCitiesFromWorldRegistry } from '../../routeModel/index.js';
```

Re-run enrichment when `world-cities.csv` changes or you need fresher World Bank years.
